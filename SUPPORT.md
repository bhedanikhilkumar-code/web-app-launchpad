# Support — Web App Launchpad

## How to Get Help

This is a portfolio/project repository. The best support path is through GitHub issues with clear context and reproducible details.

## Before Opening an Issue

Please check:

- The latest README setup instructions.
- `docs/ARCHITECTURE.md` for project structure.
- `docs/QUALITY.md` for expected checks.
- Existing issues to avoid duplicates.

## Good Issue Format

Include:

- What you were trying to do.
- What happened instead.
- Steps to reproduce.
- Screenshots, logs, or terminal output when relevant.
- Environment details such as OS, browser, device, SDK, or runtime version.

## Security Problems

Do not post secrets, tokens, exploit details, or private credentials in public issues. Use the guidance in `SECURITY.md` for responsible disclosure.

## Maintainer Expectations

This repository is maintained with a professional portfolio workflow: focused changes, documented decisions, and reviewable pull requests.
