# Code of Conduct — Web App Launchpad

## Our Standard

This project aims to be professional, respectful, and useful for learning, review, and collaboration.

Expected behavior:

- Be respectful and constructive.
- Keep feedback focused on the work, not the person.
- Assume good intent, but be clear when something needs improvement.
- Avoid harassment, insults, discrimination, spam, or hostile behavior.
- Protect private information and never request or expose secrets.

## Unacceptable Behavior

- Personal attacks or abusive language.
- Publishing private information without permission.
- Sharing malware, credential dumps, or exploit instructions in public issues.
- Repeated off-topic comments that make collaboration harder.

## Enforcement

The maintainer may edit, hide, or remove comments and issues that violate this standard. Serious or repeated violations may result in blocking from the repository.

## Reporting

If you see a problem, contact the repository owner through their GitHub profile or a trusted private channel.
